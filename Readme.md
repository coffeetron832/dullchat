## ¿Qué es dullchat?

**dullchat** es una herramienta de mensajería diseñada para que puedas hablar con quien quieras, sabiendo que lo que dices es solo para ustedes, es la solución definitiva para quienes valoran su privacidad por encima de todo. En un mundo donde tus datos son la mercancía, dullchat te devuelve el control. No hay servidores que nos vigilen, no hay bases de datos donde se guarden tus secretos.
